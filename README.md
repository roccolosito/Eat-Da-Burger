# Project Title 
### **Chomp The Burger**

## Description
An application used for logging any type of burger the user wants to devour! The app uses MySQL, Node, Express, Handlebars and a homemade ORM.

Deployed app via Heroku: https://chomptheburger.herokuapp.com/.
